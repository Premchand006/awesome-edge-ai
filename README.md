# What is Edge-AI ? 

Edge AI is the deployment and execution of artificial intelligence models directly on distributed edge devices and embedded systems for real-time, low-latency, privacy-preserving, and bandwidth-efficient perception, analytics, and autonomous decision-making near the source of data generation.
